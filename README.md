# Yuvashri Bhanuprakash - Portfolio

A modern, responsive, single-page personal portfolio website built with React, Vite, and Tailwind CSS. Featuring a data-driven architecture that separates content from UI logic for easy maintenance.

## 🚀 Live Demo
Managed and deployed via Vercel.

## 🛠️ Tech Stack
- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion (Motion)
- **Icons**: Lucide React, React Icons
- **Build Tool**: Vite

## 📁 Project Structure
- `src/data.ts`: Centralized data store for all portfolio content. Update this file to change experience, projects, or skills.
- `src/App.tsx`: Main UI component implementing a responsive layout and 3D project carousel.
- `public/`: All local image assets and logos.
